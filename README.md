# 🚗 LiDAR Obstacle Detection/LiDAR Kliūčių atpažinimas

This project develops a **basic obstacle detection algorithm** using the **KITTI dataset** and **point cloud processing** techniques. / Šis projektas sukurtas pagal **paprastą kliūčių atpažinimo algoritmą** naudojant **KITTI duomenų rinkinį** ir **taškų debesies apdorojimo** technikas. 

## 📌 Project Overview/Projekto apibendrinimas
- Process KITTI dataset's LiDAR scans / KITTI duomenų rinkinio apdorojimas atrenkant LiDAR skenavimus.
- Implement basic **object detection** (cars, pedestrians, etc.) / Paprasto **kliūčių atpažinimo** implementavimas (mašinoms, pestiesiems ir t.t.) pagal atrinktus duomenis.
- Use Open3D & PCL for **point cloud filtering and clustering** / Open3D ir PCL naudojimas **taškų debesies filtravimui ir klasterizavimui**
