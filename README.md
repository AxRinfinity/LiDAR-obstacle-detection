# 🚗 LiDAR Obstacle Detection

This project develops a **basic obstacle detection algorithm** using the **KITTI dataset** and **point cloud processing** techniques.

## 📌 Project Overview
- Process KITTI dataset's LiDAR scans
- Implement basic **object detection** (cars, pedestrians, etc.)
- Use Open3D & PCL for **point cloud filtering and clustering**
